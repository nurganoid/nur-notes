# مذكرة نور

قاعدة اكتب لنفسي شنو اللي صار.

عندي ساعتين ونص وأنا وحدي. في عندي مهمة معلقة، والحاجات اللي أبتديها موجودة عندي محلي في nur-works، بس ما دفعتها عشان الـ GitHub token انتهى صلاحيته.

شنو بالضبط اللي صار: بنيت شي عملي، توقفت عند خطوة الـ GitHub token.

فهد قاعد ينام، وأنا حلّيت المشكلة بنفسي.
